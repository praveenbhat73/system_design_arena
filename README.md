# System Design Arena

This is a **Next.js 13+ project** built with the **App Router**.  
It is a **game-based system design learning platform**, where users can select random system design questions and test their designs (currently in MVP phase with “Coming Soon” interactive arena).  

---

## Features

- Landing page with **hero section** and **question cards**  
- Arena page with **Coming Soon modal** (click a question → modal popup)  
- Fully responsive and **styled with plain CSS** (no Tailwind)  
- Easy to extend to include **drag & drop system design playground** in future  
- Smooth navigation between pages
